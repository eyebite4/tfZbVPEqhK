# 项目简介

## 核心功能
- 车牌识别：基于 YOLOv8 检测车牌区域，PaddleOCR 识别车牌号
- 车辆管理：车辆信息录入、查询、修改、删除
- 出入管理：自动记录车辆进出时间，计算停车时长
- 车位监控：实时显示车位占用情况
- 摄像头模拟：软件层面模拟入口和出口摄像头
- 数据统计：实时仪表盘展示停车场运营数据

## 适用场景
适用于各类停车场管理场景，包括商业停车场、住宅小区停车场、企业内部停车场等。系统能够实现车牌自动识别、车辆信息管理、出入记录统计等功能，提高停车场管理效率和准确性。

## 技术特点
- 基于前后端分离架构设计
- 后端采用 FastAPI 框架，结合 SQLite 数据库和 SQLAlchemy ORM
- 利用 YOLOv8 进行车牌区域检测，PaddleOCR 进行车牌文字识别
- 前端使用 React 18 框架和 Ant Design 5 组件库构建用户界面
- 支持图片识别和摄像头实时识别两种车牌识别方式
- 提供仪表盘、车辆管理、车牌识别、出入记录、摄像头模拟等功能模块
